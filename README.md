# First Aid Knowledge Base（急救知识库）

一个由个人整合制作的 PPT、文章、视频的急救学习资源库，面向志愿急救员、初学者及培训使用场景。

---

## 内容结构

本仓库分为三大核心部分：

### PPT（培训课件）
用于教学或培训场景，提供可修改源文件：

- CPR 与 AED 教学课件  
- 心律识别（VF / VT / PEA 等）  
- 出血控制与止血带使用  

路径：/PPT

---

### 文章（知识整理）
系统化整理急救知识与流程，适合查阅与复习：

- 标准操作流程（SOP）  
- 常见急症处理（低血糖、晕厥、中暑等）  
- ICS 与现场指挥逻辑  

路径：/Articles

---

### 视频（演示与教学）
用于展示操作手法或模拟场景：

- CPR / AED 操作演示  
- BVM 使用手法  
- 场景模拟（赛事、展会保障）  

路径：/Videos
